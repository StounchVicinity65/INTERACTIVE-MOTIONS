Scale Explanation

These simulations use a screen-based scale. According to CSS standards, 1 pixel is 1/96 inch, which is approximately 0.00026 meters. This scale lets us convert screen movement to real-world meters.

Why? Because p5.js and canvas animations are updated per frame, using pixels instead of physical meters. Seconds are avoided since time is frame-based (~60 frames per second).
